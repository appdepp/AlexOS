# AlexOS Cognitive Architecture v1

## Core Idea

AlexOS is not a chatbot.

AlexOS is a Cognitive Operating System.

The main intelligence is not inside the LLM.

The main intelligence is inside the Cognitive Core.

## Principle

AlexOS should not remember everything equally.

AlexOS should understand:

- what matters;
- why it matters;
- where it came from;
- how it relates to previous knowledge;
- whether it should change the model of the owner.

---

## Final Goal

AlexOS should gradually build a living model of the owner:

- goals;
- projects;
- preferences;
- habits;
- decisions;
- doubts;
- repeated patterns;
- strengths;
- weaknesses;
- thinking style;
- emotional and practical context.

The purpose is not to replace the owner.

The purpose is to help the owner think better.

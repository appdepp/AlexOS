# AlexOS Cognitive Core

## Core Idea

The intelligence of AlexOS is not the LLM.

The LLM is a language processor.

The real intelligence of AlexOS is the Cognitive Core.

---

## Cognitive Cycle

```text
Perception
    ↓
Understanding
    ↓
Memory Extraction
    ↓
Context Building
    ↓
Reasoning
    ↓
Reflection
    ↓
Memory Consolidation
```

---

## Memory Layers

- Identity Model
- Working Memory
- Episodic Memory
- Semantic Memory
- Knowledge Memory
- Decision Memory
- Vector Memory
- Graph Memory

---

## Context Kernel

AlexOS should not send all memory to the LLM.

It should build a focused context kernel for every request.

---

## Principle

Memory without understanding is storage.

Understanding without memory is temporary.

AlexOS combines both.

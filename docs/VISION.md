# AlexOS Vision

AlexOS — это персональная локальная AI-платформа, которая со временем должна стать центральным помощником владельца.

---

## Видение на 1–2 года

AlexOS должен уметь:

- работать полностью локально;
- общаться через терминал, Telegram, веб-интерфейс и голос;
- использовать локальные LLM через Ollama;
- подключаться к другим LLM при необходимости;
- хранить личную память;
- работать с документами через RAG;
- анализировать проекты, код, PDF, заметки и данные;
- помогать с GitHub;
- выполнять Python-инструменты;
- помогать с криптовалютами, яхтами, автомобилями и бизнес-идеями;
- обновлять себя из Git;
- запускать тесты;
- сообщать владельцу о состоянии системы.

---

## Главная идея

AlexOS — это не чат.

Это персональная операционная система для AI-задач.

Чат является только одним из интерфейсов.

---

## Долгосрочная цель

Владелец должен иметь возможность сказать:

> AlexOS, обнови себя.

И система должна:

1. получить последнюю версию из Git;
2. проверить зависимости;
3. запустить тесты;
4. применить обновление;
5. перезапуститься;
6. сообщить результат.

---

## Принцип развития

Каждая новая функция должна добавляться как отдельный модуль, не ломая ядро системы.
